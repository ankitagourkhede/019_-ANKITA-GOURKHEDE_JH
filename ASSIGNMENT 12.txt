1)  Write a query that selects all customers whose ratings are equal to or greater      than ANY of Serres’.
    SELECT *
    FROM customers
    WHERE rating >=ANY
    ( SELECT rating
     FROM customers
     WHERE snum IN
         ( SELECT snum
          FROM salespeople
          WHERE sname='serres' ) );

2) Write a query using ANY or ALL that will find all salespeople who have no
     customers located in their city.
    SELECT snum
    FROM salespeople
   WHERE (snum, city) NOT IN
    (SELECT snum, city
     FROM customers);

3) Write a query that selects all orders for amounts greater than any for the
    customers in London.
    SELECT *
    FROM orders
    WHERE amt> ANY
    ( SELECT amt
     FROM orders
     WHERE cnum IN
         ( SELECT cnum
          FROM customers
          WHERE city='london' ) );

4) Write the above query using MIN or MAX.
    SELECT *
    FROM orders
    WHERE amt >
    (SELECT min(amt)
     FROM customers
     NATURAL JOIN orders
     WHERE city='london');
